Question–1: What are the different ways to select an element in the DOM?

DOM can be selected in 5 ways.
getElementsByTagName()
getElementsByClassName()
getElementById()
querySelector()
querySelectorAll()

Question–2: What is the difference between innerHTML, innerText, and textContent?

innerHTML is used to set or get HTML content of an element.
innerText is used to set or get text content of an element.
innerContent is used to set or get text content of a specific node and its decendents.

Question–3: What is event delegation in the DOM?

Event delegation is the process of using event bubbling to handle events at a higher level in the DOM than the element on which the event originated.

Question–4: What is event bubbling in the DOM?

Event bubbling is a type of DOM event propagation which allows centralized event management by propagating events up the DOM tree, simplifying handling across hierarchical structures.

Question–5: How do you create, add, and remove elements using JavaScript?

this is how elements can be created, added and removed using javascript:
element = document.getElementById("btn");
const node = document.createElement("div");
node.textContent = "This is a new div";
element.appendChild(node);
